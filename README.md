# Model Evaluation Metrics

In this, I evaluated the machine learning model created in the previous task.

After training the model, I tested its performance using different evaluation metrics. These metrics helped me understand how accurately the model was making predictions.

## Tools Used

- Python
- Pandas
- Scikit-learn

## Files

- model_evaluation.py
- featured_covid_data.csv

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## What I Learned

This helped me understand the importance of evaluating a machine learning model before using it for predictions. Different metrics provide different perspectives on model performance and prediction quality.
